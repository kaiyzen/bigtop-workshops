Prezi House of Ideas


Friday September 25th, 2015 from 3pm - 10pm


Join us for the first European Bigtop Workshop.

Workshop Agenda


Time Description
3:00pm-3:15pm Welcome and Intros
3:15pm-3:45pm Bigtop: Overview and where its headed
3:45pm-4:30pm Apache Ignite: This will cover overview of Apache Ignite, the In-Memory data fabric. It will cover basic overview, how it can be used in plug/play way to accelerate your hadoop/spark jobs.
4:30pm-4:45pm Break
4:45pm-5:30pm Apache Flink: This talk will give an overview of Apache Flink, one of the newest additions to the Bigtop family. It will cover its background, general architecture and its batch and stream processing.
5:30pm-6:15pm Building a Data Product Using Apache Zeppelin: This talk will give an overview of Apache Zeppelin (incubating) - a notebook web UI built on top of Spark\Flink\Ignite. An analytics workflow will be demonstrated, resulting in a real data product built using real open dataset.
6:15pm-7:00pm Dinner and Ignite Talks
7:00pm-9:45pm Breakout and Tutorial Hacktime
9:45pm-10:00pm Wrap up and thank you's
10:00pm-? After social time at tbd..
